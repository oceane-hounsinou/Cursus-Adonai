## Description
Système de gestion numérique centralisé conçu pour automatiser le traitement des dossiers administratifs. Cette plateforme permet une gestion efficace, sécurisée et rapide des demandes, remplaçant les méthodes de traitement manuelles par une solution numérique fiable.

## Technologies Utilisées
- **Langage** : PHP
- **Base de données** : MySQL
- **Front-end** : Bootstrap, HTML5, CSS3
- **Environnement** : Développement local (XAMPP/WAMP)

## Fonctionnalités Principales
- **Authentification sécurisée** : Accès différencié pour les étudiants et les administrateurs pour garantir la confidentialité des données.
- **Automatisation** : Centralisation et suivi en temps réel du traitement des dossiers.
- **Responsive Design** : Interface adaptée pour une utilisation sur ordinateur, tablette et smartphone.
- **Gestion de contenu** : Interface d'administration pour la gestion des demandes et des utilisateurs.

## Installation
Ce projet est conçu pour être utilisé en environnement local :
1. Clonez ce dépôt sur votre machine.
2. Importez la base de données SQL incluse dans le dossier `/database`.
3. Configurez votre environnement PHP/MySQL (via XAMPP ou WAMP).
4. Accédez au projet via votre navigateur (ex: `http://localhost/gestdemande`).
